# Code Index

This directory contains lightweight runnable demos aligned with the collected paper topics.

## satellite_fl_client_selection

- Paper theme: client selection in satellite federated learning
- Script: `client_selection_demo.py`
- Purpose: simulates visibility-aware client selection under contact, energy, and fairness constraints

## multi_satellite_mission_planning

- Paper theme: adaptive teaching-learning-based optimization for multi-satellite planning
- Script: `tlbo_demo.py`
- Purpose: runs a small discrete optimizer for assigning observation tasks to multiple satellites under capacity constraints
